# fbm3d
fbm3d is a code that generates a lognormal or Gaussian random field using the fractional Brownian motion algorithm, designed to mimic the density field in the interstellar medium.
The detailed algorithms are described in [Seon (2012)](https://ui.adsabs.harvard.edu/search/q=author%3A%22Seon%22+year%3A2012+fBm) and [Lewis & Austin (2002)][lewis-austin-2002].

See [example.ipynb](example.ipynb), [example2.ipynb](example2.ipynb), and [example3.ipynb](example3.ipynb) for examples of how to use the code.

[lewis-austin-2002]: https://ams.confex.com/ams/pdfpapers/4
